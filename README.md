# libds
`libds` is a minimal and fast data structure library written in C.  
Designed for system-level projects and educational use.

## Data Structures

1. Queue 
    - void initQueue(queue *q);
    - int isEmpty(queue *q);
    - int isFull(queue *q);
    - void enqueue(queue *q, int value);
    - int dequeue(queue *q);
    - void printq(queue *q);
